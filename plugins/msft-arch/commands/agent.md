name: agent
description: Launch the Microsoft Architecture orchestrator agent to route your request to the right specialized skill
---

Invoke the agent to help with Microsoft enterprise architecture tasks. The agent will analyze your request and route it to the appropriate specialized skill:

**Process**: discover, stack-select, requirements, validate
**Specialists**: azure-architect, powerplatform-architect, d365-architect, container-architect, data-architect, ai-architect
**Output**: spec, artifacts, docs
**Odin**: odin
**TOM**: tom-architect

Usage: `/msft-arch:agent <describe your architecture need>`
