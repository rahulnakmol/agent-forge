Invoke Odin for deep reasoning and expert architecture guidance. Odin analyzes your code and designs without making changes, returning structured advisory output.

**Capabilities:**
- Architecture review and design validation
- Security assessment (STRIDE analysis, threat modeling)
- Impact analysis for proposed changes
- Legacy code comprehension
- Complexity decomposition and simplification
- Technology decision trade-off analysis

**Usage:**
```
/msft-arch:odin Review the architecture in design.md
/msft-arch:odin How should I build a claims processing system?
/msft-arch:odin Simplify this microservice — it has 47 endpoints
/msft-arch:odin Assess the security of our API gateway design
```

Odin returns a structured analysis (Assessment, Recommendations, Risks, Next Steps) that you or other agents can act upon.
