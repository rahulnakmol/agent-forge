Governance

Functional process
The leading practices, process taxonomies, process flows and maturity models that describe functional operation. The backbone of the Target Operating Model to which everything else relates.
Service delivery model
The model that defines where and by whom processes should be performed. It also describes incremental components required to run and operate a capability center.
People
The roles, organizational relationships, competencies and skill requirements needed to execute process areas. This layer intersects with the functional process to describe who does what.
