> **Architect Notes**: The KPIs Linked to Benchmarks asset is a comprehensive list of key indicators, including Key Performance Indicators, Process Performance Indicators, and Key Risk Indicators. Key Performance Indicators measure the outcomes of a business process, while Process Performance Indicators measure the efficiency or performance of a business process. Key Risk Indicators measure the effectiveness of key controls to ensure proper monitoring. Together, these help us identify which KPIs are areas of focus for transformation so that we can measure and set performance improvement targets. In this example, we can see the process and name of each KPI. To the right, the formula and business purposes are described. Moving further to the right, we can see unit of measure indicated. If there are APQC licensed data for specific KPIs, then it’s indicated in this column, and the URL for that benchmark data can be found in this column. At this time for Sustainability, there are no KPIs aligned to APQC.
