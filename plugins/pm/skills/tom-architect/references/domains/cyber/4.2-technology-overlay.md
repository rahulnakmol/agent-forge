> **Architect Notes**: The Supporting Technology Overlay is a tech specific asset. It is an end-to-end process taxonomy that shows the underlying technology applications that enable each level 3 process. It gives us details on the application modules we will need to best enable the end-to-end process. Our platform-specific assets are enabled through a variety of technology platforms, including Oracle, SAP, Workday, Microsoft, and other leading vendors. In this example, we can explore the Oracle tools that can optimize the Acquire to Retire process flows by viewing this supporting technology overlay. The “Create Book/Tax Asset Record & Componentize” process is performed in Oracle Cloud Fixed Assets, while the “Maintain Internal Controls” process is performed in Oracle RMCS. The “Process Intercompany Transfers” process is performed using two systems: Oracle Cloud GL and Oracle Cloud Fixed Assets.
