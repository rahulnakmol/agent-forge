## Application Architecture, Data Flow Diagram and Integrations List

> **Architect Notes**: The Application Architecture, Data Flow Diagram and Integrations List is a tech specific asset that describes a client's applications, data integration, and data flows. This asset connects the process inputs to the applications and reporting. It also shows how metadata and transaction data flow between applications, and how the applications and data flow into reporting outputs. The Application Architecture, Data Flow Diagram and Integrations List provides an easy-to-understand architecture diagram that the business can understand while also providing a very detailed view for IT.
